# /biz-plan

Gather context, then generate a business plan.

## Step 1 — Gather Context

Ask the user:
1. What is the business / product?
2. What stage are you at? (idea / early / revenue / growth)
3. Who is the primary audience for this plan? (investors / internal / bank / co-founder)
4. What's the time horizon? (1 year / 3 year / 5 year)

## Step 2 — Generate Business Plan

```markdown
# Business Plan: [Company / Product Name]
Date: [Date] | Stage: [Stage]

## Executive Summary
[2–3 paragraphs: what you do, for whom, why it will win]

## Problem & Opportunity
- The problem: [specific, painful, measurable]
- Market context: [why this problem is growing]
- Why now: [what's changed that makes this possible]

## Solution
- What we offer: [clear, jargon-free description]
- How it works: [simple flow]
- Why it's better: [key differentiators from status quo]

## Market Analysis
- TAM: $X [source / methodology]
- SAM: $X [based on: geography, segment]
- SOM (3-year): $X [realistic capture]
- Target segment: [who exactly, and why start here]

## Competitive Landscape
[2–3 sentence overview + positioning table]

## Business Model
- Revenue model: [how you make money]
- Pricing: [tiers or price points]
- Unit economics: [CAC, LTV, payback — estimated if early]

## Go-to-Market Strategy
- Phase 1: [First channel / customer / segment]
- Phase 2: [Expansion]
- Phase 3: [Scale]

## Operations Plan
- Key activities: [what the business does every day]
- Key resources: [people, tech, IP, partnerships]
- Key milestones: [6 months / 12 months / 24 months]

## Team
[Founders + key hires + advisors]

## Financial Projections
| Metric | Month 6 | Year 1 | Year 2 | Year 3 |
|--------|---------|--------|--------|--------|
| Revenue | | | | |
| Costs | | | | |
| EBITDA | | | | |
| Customers | | | | |

## Funding Ask (if applicable)
- Raising: $X
- Use of funds: [breakdown %]
- Milestones this achieves: [list]
- Runway: [X months]

## Risks & Mitigations
| Risk | Probability | Impact | Mitigation |
|------|-------------|--------|------------|
| | | | |
```
