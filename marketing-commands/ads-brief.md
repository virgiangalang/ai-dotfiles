description: Create a complete paid ads campaign brief — campaign structure, targeting, creative brief, bidding strategy, and success metrics for Google, Meta, or TikTok.
---

Load the `ads-performance` skill and activate the `ads-manager` agent.

Create a complete paid ads campaign brief. Include:
1. **Campaign Goal** — what is the conversion event? (lead, trial, purchase, call)
2. **Target CPL / CPA / ROAS** — define the number before the campaign launches
3. **Platform** — Google Search / Meta / TikTok / all three? (platform choice must justify ICP's behavior)
4. **Campaign Structure** — Campaign → Ad Set → Ad hierarchy with settings
5. **Audience** — targeting approach (interest / lookalike / custom / broad)
6. **Creative Brief** — 3 hook variants, format (image/video/carousel), CTA options
7. **Budget** — daily / total, split between learning phase and scaling phase
8. **Tracking** — conversion events, UTM parameters, attribution setup
9. **Optimization Schedule** — when to review, what metrics trigger changes

Apply the Quality Gate from the ads-performance skill before delivering. Flag any missing tracking or creative requirements.
