description: Review marketing copy (ads, landing pages, email, social) against conversion best practices, brand voice, and framework standards. Delivers specific, actionable feedback.
---

Load the `copywriting` skill and activate the `copywriter` agent.

Review the provided marketing copy. Evaluate against:
1. **4 U's Test** (Urgent / Unique / Useful / Ultra-specific) — score each headline, flag scores below 3/4
2. **Opening Line** — does it start with the reader's problem or outcome? Flag "We are..." or "Our product..." openings
3. **Clarity** — can a first-time reader understand the offer and benefit in 10 seconds?
4. **CTA** — is there exactly one CTA? Is it specific and benefit-driven?
5. **Brand Voice** — does it match the brand voice guide? (Ask for the voice guide if not provided)
6. **Framework Check** — does the structure follow AIDA, PAS, or another proven framework?
7. **Social Proof** — is there evidence (testimonials, numbers, case studies) where appropriate?

Deliver feedback with:
- **CRITICAL**: Blocking issues (missing CTA, wrong audience, off-brand)
- **HIGH**: Conversion killers (weak headline, vague CTA, no proof)
- **MEDIUM**: Improvements (stronger hook, additional variant, A/B test suggestion)
- **Rewrite**: For CRITICAL and HIGH issues, provide a rewrite, not just feedback
