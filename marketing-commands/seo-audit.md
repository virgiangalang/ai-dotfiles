description: Conduct an SEO audit covering technical health, on-page optimization, keyword gaps, and content opportunities. Delivers a prioritized fix list with expected impact.
---

Load the `seo-strategy` skill and activate the `content-strategist` agent.

Conduct a comprehensive SEO audit. Cover all four layers:

**1. Technical SEO**
Apply the technical SEO audit checklist from the seo-strategy skill:
- Crawlability: robots.txt, sitemap, orphan pages, redirect chains
- Indexability: noindex errors, canonicals, duplicate content
- Site speed: Core Web Vitals (LCP, CLS, FID) via PageSpeed Insights
- Architecture: click depth, URL structure, internal linking
- Mobile: responsive design, font size, touch targets

**2. On-Page SEO**
For the top 10 priority pages:
- Title tag (50–60 chars, includes keyword)
- Meta description (150–160 chars, includes CTA)
- H1 (one per page, contains primary keyword)
- Keyword usage in first 100 words, H2s, image alt text
- Internal linking from high-authority pages

**3. Keyword Gap Analysis**
- What keywords are competitors ranking for that we're not?
- What keywords do we rank in position 4–20 (quick-win optimization targets)?
- What high-intent keywords are not covered by any existing page?

**4. Content Opportunities**
- Top 5 articles to create (high volume, low competition, commercial intent)
- Top 5 existing articles to update (ranking 11–30, content outdated)
- Pillar page gaps (is there a comprehensive resource for each content pillar?)

Deliver findings as:
- **CRITICAL**: Technical issues blocking indexing
- **HIGH**: On-page issues on high-traffic pages
- **MEDIUM**: Keyword gaps and content opportunities
- Prioritized action list with estimated effort (Low / Medium / High) and expected impact
