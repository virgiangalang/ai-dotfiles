---

Load the `content-strategy` skill and activate the `content-strategist` agent.

Build a detailed content calendar. Deliver:
1. **Content Pillars** (if not defined) — 3–5 topic areas that align to brand + ICP need + keyword opportunity
2. **Content Calendar** — 4–8 week calendar with these fields per piece:
   - Week / publish date
   - Pillar (1–5)
   - Format (blog / video / LinkedIn / email / social / etc.)
   - Working title
   - Target keyword (for SEO content)
   - Funnel stage (TOFU / MOFU / BOFU)
   - CTA (one specific action)
   - Channel(s)
   - Owner
   - Status (brief / draft / scheduled)
3. **Content Mix** — confirm 40% educational / 30% MOFU / 20% BOFU / 10% community
4. **Distribution Plan** — how each piece gets distributed after publishing
5. **Success Metrics** — organic traffic target, leads from content, keyword ranking targets

Apply the Quality Gate from content-strategy skill. Ask for existing content and keyword research before building if not provided.
