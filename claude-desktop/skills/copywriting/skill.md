# Copywriting

## Skill Stack

- `brand-strategy` — every copy output must match the brand voice and messaging hierarchy
- `ads-performance` — ad copy is the creative layer of paid campaigns; brief the copy before the creative
- `email-marketing` — email copy uses different frameworks than landing page copy; context determines format
- `content-strategy` — long-form content copy follows the content brief
- `market-research` (ECC) — the ICP's language is the best source of copy; use customer interview quotes

## Guardrails

- Copy that sounds clever but doesn't communicate a benefit is decoration — clarity beats creativity
- The headline is 80% of the work — if the headline doesn't hook, the rest is unread
- Never start copy with "We are..." or "Our product..." — start with the reader's problem or desired outcome
- Every piece of copy needs one CTA — two CTAs halve conversion; more than two is noise
- Do not write copy before reading existing customer reviews, interviews, or testimonials — the customer's language is better than yours

## Framework 1: AIDA (Awareness → Interest → Desire → Action)

For landing pages, sales pages, email campaigns.

```
ATTENTION: Stop the scroll / grab the eye
  → Headline that speaks to the pain or outcome

INTEREST: Make them want to keep reading
  → Expand the problem or promise; prove you understand them

DESIRE: Make them want what you're selling
  → Specific benefits (not features), social proof, results

ACTION: Tell them exactly what to do next
  → Single, clear, low-friction CTA

AIDA TEMPLATE:
──────────────────────────────────────────────
[HEADLINE]: [Pain / Promise / Curiosity hook]

[INTEREST COPY]: If you're [experiencing this situation], you know how frustrating
it is when [specific pain]. Most [ICP] struggle with [problem] because [reason].

[DESIRE COPY]: [Product] helps you [achieve outcome] by [mechanism].
[Customer name] used it to [specific result in specific timeframe].
[Social proof / logo strip / rating]

[CTA]: [Action verb] + [outcome] → [Button: "Get [result]" / "Start [free/now]"]
```

## Framework 2: PAS (Problem → Agitate → Solution)

For ad copy, short-form social, email subject lines, sales pages.

```
PROBLEM: Name the pain clearly.
  "Most [ICP] spend hours every week on [task] — and it still doesn't work."

AGITATE: Make the pain feel real and urgent.
  "That's [X hours] per month you'll never get back. Time that should go to
  [what they actually care about]. And if you keep doing it the same way,
  [consequence of inaction]."

SOLUTION: Present your product as the relief.
  "That's why we built [Product]. [One-sentence description].
  [Specific result]. [CTA]."
```

## Framework 3: 4 U's (Urgent · Unique · Useful · Ultra-specific)

Use this as a checklist for any headline or CTA.

```
Urgent:         Does it create a reason to act now?
Unique:         Does it say something competitors don't?
Useful:         Does it promise a benefit the reader actually wants?
Ultra-specific: Is it specific enough to be believable?

WEAK: "Improve your marketing"
STRONG: "Get 3x more leads from your existing budget in 30 days"

4 U's Score:
  [Urgent: Y/N] [Unique: Y/N] [Useful: Y/N] [Ultra-specific: Y/N]
Target: 3+ out of 4
```

## Headline Formulas

```
TYPE 1 — NUMBER + BENEFIT
"[Number] [Ways / Secrets / Steps] to [Desired Outcome]"
"7 Ways to Double Your Email Open Rate"

TYPE 2 — HOW TO
"How to [Achieve Goal] Without [Common Obstacle]"
"How to Scale Paid Ads Without a $50K Budget"

TYPE 3 — PROBLEM → SOLUTION
"[Common Problem]? Here's How to Fix It in [Timeframe]"
"Churn Rate Too High? Here's How to Reduce It by 40% in 60 Days"

TYPE 4 — RESULT + TIMEFRAME
"[Specific Result] in [Specific Timeframe]"
"From 0 to $10K MRR in 90 Days"

TYPE 5 — QUESTION (curiosity)
"Are You Making These [X] [Topic] Mistakes?"
"What If You Could [Desired Outcome] Without [Pain]?"

TYPE 6 — FEAR / RISK
"Stop [Doing This Wrong Thing] Before It Costs You [Specific Loss]"
"The Hiring Mistake That's Quietly Destroying Your Team Culture"

TYPE 7 — SOCIAL PROOF
"How [Real Customer] [Achieved Outcome] with [Product]"
"How 3,000+ Founders Use [Product] to [Key Benefit]"
```

## Landing Page Copy Structure

```
ABOVE THE FOLD (must communicate value in 5 seconds)
  H1: [Primary benefit headline — big, bold, specific]
  H2/Subheadline: [Expand on H1 — who it's for + how it works]
  CTA Button: "[Action verb] [outcome]" (e.g., "Start Free Trial", "Get the Report")
  Social proof line: "Join 5,000+ [ICP] who [key result]" or trust logos

PROBLEM SECTION
  [Empathy paragraph — "You know what it's like when..."]
  [Pain points in bullets — 3-5 pains the ICP lives with]

SOLUTION SECTION
  [Product introduction — what it is + what it does]
  [Key features → framed as benefits: Feature enables → Benefit means you can]
  Feature 1 → Benefit 1
  Feature 2 → Benefit 2
  Feature 3 → Benefit 3

PROOF SECTION
  [Customer testimonials — include name, title, company, photo]
  [Case study: Before / After / Results]
  [Metrics: "X customers / $X saved / X% improvement"]

OBJECTION HANDLING
  [FAQ — top 3-5 objections in Q&A format]

FINAL CTA
  [Restate the benefit + urgency if applicable]
  [Button] [Risk removal: "No credit card required" / "Cancel anytime" / "Free for 14 days"]
```

## Ad Copy Templates

**Short-form (social ads — 125 chars):**
```
[Pain hook]: "[Problem statement]? [Promise]."
"Stop guessing which ads work. See exactly what's driving revenue."

[Result lead]: "[Specific result] in [timeframe]. No [obstacle]."
"10x your organic reach in 30 days. No paid ads required."

[Question]: "What if [desired outcome] was [easier/faster/cheaper]?"
"What if closing enterprise deals took 2 weeks, not 3 months?"
```

**Email Subject Lines:**
```
Curiosity: "The [topic] mistake costing you [loss]"
Result: "How [customer] got [result] in [timeframe]"
Personal: "Quick question about [their company/role/pain]"
Number: "[X] ways to [outcome] this week"
Fear: "Don't make this [topic] mistake"
Direct: "[Specific offer / what's inside]"
```

## CTA Best Practices

```
WEAK CTAs: "Submit" / "Click Here" / "Learn More" / "Sign Up"
STRONG CTAs: Specific action + specific outcome

Formula: [Verb] + [what they get]
"Get My Free Analysis"
"Start Saving 5 Hours a Week"
"See [Product] in Action"
"Claim Your Spot"
"Download the Playbook"

Button copy rules:
- First person performs better: "Start MY trial" > "Start YOUR trial"
- Outcome over action: "Get better leads" > "Submit"
- Risk reduction beside button: "No credit card · Cancel anytime · Free for 14 days"
```

## Quality Gate

Before finalizing any piece of copy:
- [ ] Headline tested against 4 U's checklist (score 3+/4)
- [ ] Copy does NOT open with "We are..." or "Our product..." — opens with reader's problem or outcome
- [ ] Benefits stated, not just features (every feature translated to "which means...")
- [ ] Single CTA only
- [ ] Voice and tone matches brand voice guide
- [ ] Customer language used (from actual reviews, interviews, or testimonials)
- [ ] Copy read aloud — does it flow naturally?
- [ ] A/B variant created for headline and CTA (minimum)
