# Brand Strategy

## Skill Stack

- `market-research` (ECC) — understand competitor positioning before defining yours; find the white space
- `competitive-platform-analysis` (ECC) — audit competitor brand messaging, visual identity, and positioning claims
- `marketing-strategy` — brand strategy must precede channel and campaign decisions
- `copywriting` — messaging framework becomes the input for all copy
- `stakeholder-comms` — brand guidelines presentation to board or leadership

## Guardrails

- Positioning is about choice — a brand that tries to be everything to everyone is nothing to anyone; force the tradeoffs
- Never write the positioning statement before completing competitive analysis — you need the white space first
- Brand voice is not "professional and friendly" — every brand says that; specificity is what makes it real (name brands that sound like you, name brands that don't)
- Messaging hierarchy must flow: positioning → pillars → proof points → copy — never bottom-up
- A rebrand that doesn't address the underlying strategic problem (wrong ICP, unclear value prop, market shift) will fail — brand cannot save a broken business model

## Mode 1: Brand Positioning

**Positioning Statement Formula:**
```
For [target customer],
[Brand Name] is the [category/frame of reference]
that [key benefit — the job you do for them]
because [reason to believe / proof].

Example:
For early-stage B2B SaaS founders,
Lunara is the strategic reflection tool
that helps founders see their business clearly
because it combines structured frameworks with AI-powered context memory.
```

**Positioning Map (2×2 matrix):**
Choose the two axes that matter most in your market (price/quality, speed/depth, enterprise/consumer, etc.). Plot yourself and all competitors. Your position should be in uncontested or underserved space.

**Positioning Checklist:**
- [ ] Does it name a specific customer?
- [ ] Is it a real category (not "a platform for everything")?
- [ ] Does it name one primary benefit?
- [ ] Is the reason to believe provable?
- [ ] Could a competitor honestly say the same thing? (If yes, rewrite)

## Mode 2: Brand Pillars

Brand pillars are the 3–5 core attributes that every product, campaign, and communication must embody.

**Pillar Template:**

```
PILLAR 1: [Name — one word or short phrase]
Definition: [What this means for us]
In practice: [What we do because of this]
We would never: [What this rules out]

PILLAR 2: [Name]
...

PILLAR 3: [Name]
...
```

**Examples by brand archetype:**

| Archetype | Brand Pillars |
|-----------|--------------|
| Disruptor | Bold, Honest, Unconventional |
| Expert | Rigorous, Authoritative, Precise |
| Companion | Warm, Reliable, Accessible |
| Innovator | Curious, Forward-looking, Experimental |
| Guardian | Trustworthy, Transparent, Protective |

## Mode 3: Brand Voice & Tone

Brand voice is WHO you are. Tone is HOW you sound in a specific context (celebrating a win vs. apologizing for an outage).

**Voice Spectrum (pick 3 attributes, define the range):**

```
We are:              We are NOT:
[Attribute]          [The negative extreme of that attribute]
─────────────────────────────────────────────────
Bold                 Arrogant
Warm                 Sycophantic
Direct               Abrasive
Expert               Condescending
Playful              Immature
Honest               Blunt to the point of cruelty
```

**Tone by context:**

| Context | Tone Adjustment |
|---------|----------------|
| Product launch | Excited, proud, forward-looking |
| Error / outage | Direct, responsible, no excuses, action-focused |
| Customer success story | Warm, celebratory, customer-centric |
| Thought leadership | Authoritative, evidence-backed, opinionated |
| Social media | More casual, conversational, humanized |
| Legal / compliance | Precise, clear, still on-brand where possible |

**Voice Examples:**

```
Topic: We made an error in billing.

OFF-BRAND (generic): "We regret to inform you that an error occurred."

ON-BRAND (example — direct, honest, warm):
"We made a mistake. Your billing was charged incorrectly this month,
and we've already fixed it and refunded the difference. Here's exactly
what happened and what we've changed to make sure it doesn't happen again."
```

## Mode 4: Messaging Hierarchy

The messaging hierarchy ensures every piece of communication reinforces the same core story, from elevator pitch to landing page to sales deck.

```
MESSAGING HIERARCHY
─────────────────────────────────────────────
BRAND PROMISE (one sentence — the overarching commitment)

HEADLINE MESSAGE (what we lead with — the hook)

VALUE PROPOSITION (what we do + for whom + the outcome)

PILLAR 1 MESSAGE:
  Claim: [what we say]
  Proof: [evidence — case study, data, feature]

PILLAR 2 MESSAGE:
  Claim:
  Proof:

PILLAR 3 MESSAGE:
  Claim:
  Proof:

OBJECTION HANDLING:
  Objection 1: [most common reason they don't buy]
  Response: [how we address it]

  Objection 2:
  Response:

TAGLINE OPTIONS:
  1. [Option — tested]
  2. [Option — aspirational]
  3. [Option — functional]
```

## Mode 5: Brand Audit

Use when evaluating whether current brand is consistent and effective.

**Brand Audit Checklist:**

```
CONSISTENCY
- [ ] Logo used correctly across all touchpoints?
- [ ] Brand colors consistent (no off-brand shades)?
- [ ] Typography consistent?
- [ ] Voice and tone consistent across website, social, email, sales?

CLARITY
- [ ] Can a prospect understand what you do in 5 seconds from the homepage?
- [ ] Is the ICP clearly identifiable from the messaging?
- [ ] Is the primary benefit clear?

DIFFERENTIATION
- [ ] Does the positioning differ from top 3 competitors?
- [ ] Is there a clear reason to believe?
- [ ] Are there any positioning claims that are generic ("innovative", "reliable")?

EMOTIONAL CONNECTION
- [ ] Does the brand evoke a clear feeling?
- [ ] Would a customer describe the brand in the same words we use internally?
- [ ] Is the brand memorable (would someone recognize it out of context)?
```

## Quality Gate

Before presenting brand strategy as complete:
- [ ] Competitive landscape mapped — positioning white space identified, not assumed
- [ ] Positioning statement written using the formula (For / is the / that / because)
- [ ] 3–5 brand pillars defined with "we would never" examples
- [ ] Voice attributes (3) defined with positive + negative range
- [ ] Messaging hierarchy complete from promise → pillars → proof points → objections
- [ ] No generic claims ("innovative", "world-class", "seamless") without a proof point backing them
- [ ] A real human from the target ICP has reviewed and confirmed the messaging resonates
